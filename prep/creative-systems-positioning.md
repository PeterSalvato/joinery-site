# Positioning: Joinery as a School for Creative Systems

**Status:** STAGED / PREP
**Objective:** Reposition Joinery around "Creative Systems" and "Modern Tooling," moving AI from the subject to the engine.

---

## 1. The Core Definition
**Joinery is a school for building creative systems.** 
We teach creative practitioners how to use AI and modern tooling to build the infrastructure of their own practice. We don't teach "prompts"; we teach **Creative Systems Engineering.**

## 3. The "Creative Systems" Manifesto (Homepage)

> "Creative work is currently trapped in a prompting loop. You talk to a model, you get a guess, you tweak the words, you try again. 
> 
> **Joinery is for the practitioners who want to build the system instead of fighting the prompt.**
> 
> We teach a structural methodology for building creative systems using AI and modern tooling. We don't start with what you say; we start with how the system is built. When the architecture is right, the AI provides the depth and the system provides the coherence."
